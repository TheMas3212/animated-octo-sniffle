NodeStats interface

Object representing all the values related to a hacknet node.

Signature:

export interface NodeStats

Properties

Property Type Description
cache? number (Optional) Cache level. Only applicable for Hacknet Servers
cores number Node's number of cores
hashCapacity? number (Optional) Hash Capacity provided by this Node. Only applicable for Hacknet Servers
level number Node's level
name string Node's name
production number Node's production per second
ram number Node's RAM (GB)
ramUsed? number (Optional) Node's used RAM (GB)
timeOnline number Number of seconds since Node has been purchased
totalProduction number Total number of money Node has produced
